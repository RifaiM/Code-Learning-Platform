Event Handling Concepts Explained

üéØ addEventListener Method

addEventListener() is the modern, flexible way to attach event handlers. It allows multiple listeners for the same event type.

// Basic syntax
element.addEventListener("eventType", functionToRun);

// Example with click event
let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("Button was clicked!");
});

üñ±Ô∏è Click Event Handling

Click events are triggered when a user clicks on an element. They're the most common type of user interaction.

// HTML: <button id="colorBtn">Change Color</button>
// JavaScript: Handle click events
let colorButton = document.getElementById("colorBtn");

colorButton.addEventListener("click", function() {
    document.body.style.backgroundColor = "lightblue";
    console.log("Background color changed!");
});

üèÉ Hover Events (mouseover/mouseout)

Hover events respond when the mouse cursor enters or leaves an element area.

// Handle hover events
let hoverElement = document.getElementById("hoverBox");

hoverElement.addEventListener("mouseover", function() {
    this.style.backgroundColor = "yellow";
    this.innerHTML = "Mouse is over me!";
});

hoverElement.addEventListener("mouseout", function() {
    this.style.backgroundColor = "white";
    this.innerHTML = "Hover over me!";
});

üé® Multiple Event Types

You can attach multiple different event listeners to the same element for rich interactions.

// Multiple events on one element
let interactiveBox = document.getElementById("multiBox");

interactiveBox.addEventListener("click", function() {
    this.style.borderColor = "red";
});

interactiveBox.addEventListener("mouseover", function() {
    this.style.transform = "scale(1.1)";
});

interactiveBox.addEventListener("mouseout", function() {
    this.style.transform = "scale(1)";
});
