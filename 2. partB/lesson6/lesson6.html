Lesson 6: CSS Basics

Introduction

CSS (Cascading Style Sheets) is the language that controls how HTML elements look on a webpage. While HTML provides the structure and content, CSS handles the visual presentation - colors, fonts, spacing, and layout.

CSS Syntax

selector {
  property: value;
  property: value;
}

Your Task

Create a styled webpage using CSS. Your page should include:

  • Set the color property for at least one element
  • Apply a font-family to text elements
  • Use a background color or style
  • Include both element selectors (like h1) and class selectors (like .highlight)
  • Use proper CSS syntax with selectors, properties, and values

Validation requirements: Must contain color, font-family, and background properties with proper CSS syntax.
