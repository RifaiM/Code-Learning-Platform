CSS Box Model Demo

Each element below demonstrates different margin, padding, and border configurations:

Box 1 - Standard Spacing

This box has uniform margin (20px), padding (15px), and a solid border.

Box 2 - Asymmetric Spacing

This box has different horizontal and vertical margins and padding values.

Box 3 - Centered with Rounded Border

Box Model Demonstration

  • Margin: Space outside the border
  • Border: The border around the element
  • Padding: Space inside the border
  • Content: The actual content area
