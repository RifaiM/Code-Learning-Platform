Lesson 1 — HTML Basics & Website Structure

Introduction

HTML (HyperText Markup Language) is the standard language for creating webpages. It structures your content and tells the browser what each part means.

HTML is made of tags . A tag usually has an opening tag and a closing tag . For example: 

<p>This is a paragraph</p>
  • Opening tag: <p>
  • Content: The text inside the tag.
  • Closing tag: </p> (note the slash / ).

Some tags are self-closing (no content inside), like <br> for a line break or <img> for an image.

Tags can also have attributes to add extra information. Example: 

<img src="photo.jpg" alt="A beautiful view">
