üì¶ Variables & Data Storage

Variables are containers that store data values. Use let for changeable values and const for constants.

const PI = 3.14159;
let score = 100;

üî¢ Operators & Calculations

Operators perform operations on variables and values. JavaScript supports arithmetic, comparison, and logical operators.

let isEqual = x === y;   // Comparison: true/false
