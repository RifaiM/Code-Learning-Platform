Lesson 3: Links & Images

Introduction

Links and images are essential elements that make websites interactive and visually appealing. The <a> tag creates links to other pages or websites, while the <img> tag displays images.

Links - The <a> Tag

Links use the href attribute to specify where they go:

<a href="https://example.com">Visit Example</a>
  • href: The destination URL or file path
  • target="_blank": Opens link in a new tab

Images - The <img> Tag

Images are self-closing tags with important attributes:

<img src="photo.jpg" alt="Description of photo" width="200">
  • src: The image file path or URL
  • alt: Alternative text for accessibility (required!)
  • width/height: Control image size
