Lesson 4: Lists & Navigation

Introduction

Lists help organize information in a structured way, while navigation elements guide users through your website. HTML provides different types of lists and semantic elements for creating clear, accessible navigation.

Types of Lists

📋 Unordered List (<ul>)

For items without a specific order:

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>

🔢 Ordered List (<ol>)

For items with a specific sequence:

<ol>
  <li>Open the app</li>
  <li>Sign in</li>
  <li>Start coding</li>
</ol>

Navigation Element

The <nav> element is used to wrap navigation links, making your website more semantic and accessible:

<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>

Understanding List Structure

Lists must follow a specific structure for proper HTML:

🏗️ Container Element
<ul> or <ol> wraps all items
📄 List Items
Each <li> contains one item
🧭 Navigation
<nav> element for semantic meaning

✅ Correct Structure:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

❌ Incorrect Structure:

<ul>
  Item 1
  Item 2
</ul>

⚠️ List items must be wrapped in <li> tags!
